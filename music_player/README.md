# Music Player 🎵

Интерактивный музыкальный плеер с клавиатурным управлением.

## Запуск

```bash
pip install pygame
python main.py
```

Положи `.wav` или `.mp3` файлы в папку `music/` перед запуском.

## Управление

| Клавиша | Действие           |
|---------|--------------------|
| P       | Воспроизвести      |
| S       | Стоп               |
| SPACE   | Пауза / Продолжить |
| N       | Следующий трек     |
| B       | Предыдущий трек    |
| Q / ESC | Выход              |

## Ключевые концепции

- `pygame.mixer.init()` — инициализация звуковой системы
- `pygame.mixer.music.load(path)` — загрузка трека
- `pygame.mixer.music.play()` — воспроизведение
- `pygame.mixer.music.get_pos()` — позиция в мс
- `event.type == pygame.KEYDOWN` — обработка нажатий
