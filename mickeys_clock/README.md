# Mickey's Clock

Часы с руками Микки Мауса.

## Запуск
```bash
python3 main.py
```

## Управление
| Клавиша | Действие |
|---------|----------|
| ESC     | Выход    |

## Настройка размера рук
В `main.py` измени:
```python
HAND_H = 170  # высота руки (уменьши если слишком длинные)
HAND_W = 35   # ширина руки
```
